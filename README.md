# GE Offer Timer

A RuneLite plugin that displays how long each Grand Exchange offer has been active, including buy/sell status and fill progress.

## Features

- ⏱️ **Offer Duration** — Shows elapsed time for each active GE offer
- 📊 **Fill Progress** — Displays how much of each offer has been filled
- 🟢🔴 **Buy/Sell Status** — Clearly indicates whether each offer is a buy or sell

## Installation

This plugin is available on the [RuneLite Plugin Hub](https://runelite.net/plugin-hub).

1. Open RuneLite
2. Click the Plugin Hub icon (plug icon at the top of the configuration panel)
3. Search for **GE Offer Timer**
4. Click **Install**

## Use Cases

- Track how long your flips have been sitting so you know when to adjust prices
- Monitor fill progress across all 8 GE slots at a glance
- Quickly identify stale offers that need repricing
