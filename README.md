# GE Offer Timer
A RuneLite plugin that displays how long each Grand Exchange offer has been active, including buy/sell status and fill progress.

## Features
- ⏱️ **Offer Duration** — Shows elapsed time for each active GE offer
- 📊 **Fill Progress** — Displays filled/total quantity for each offer
- 🟢🟡 **Buy/Sell Status** — Color coded: green for buys, yellow for sells
- ✅ **Completed Offers** — Frozen timer shown for bought/sold/cancelled offers
- ⚙️ **Only Show at GE** — Optional toggle to only show the overlay when the GE is open
- 💾 **Persistent Timers** — Timers survive logouts and restarts

## Installation
This plugin is available on the RuneLite Plugin Hub.
1. Open RuneLite
2. Click the Plugin Hub icon
3. Search for **GE Offer Timer**
4. Click Install

## Use Cases
- Track how long your flips have been sitting so you know when to adjust prices
- Monitor fill progress across all 8 GE slots at a glance
- Quickly identify stale offers that need repricing

## Building
```bash
./gradlew build
```
